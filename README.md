# CSS Grid — Galerie de produits

Mini-projet d'apprentissage du CSS Grid : galerie de produits avec page de détail.

**Stack :** HTML · CSS Grid · JavaScript Vanilla

---

## Pages

| Fichier | Description |
|---------|-------------|
| `index.html` | Galerie principale (CSS Grid) |
| `detail.html` | Page de détail d'un produit |
| `style.css` | Mise en page CSS Grid |
| `script.js` / `detail.js` | Interactions JavaScript |

---

## Lancer le projet

Ouvre simplement `index.html` dans un navigateur — aucune installation requise.

---

## Objectif pédagogique

- Maîtriser la mise en page avec `display: grid`
- Pratiquer la navigation entre pages en JavaScript vanilla
